**Excel AutoReport** is an AI-powered automation agent designed to streamline the reporting process for business data. By combining the vast analytical capabilities of `pandas` with the advanced reasoning of LLMs (powered by Groq and Llama 3.3 70B), this tool completely automates the extraction, analysis, and summarizing of Excel data into professional, ready-to-present Markdown business reports.

Say goodbye to manual data crunching and hello to instant insights! ✨

---

## 🚀 Key Features

- **Automated Data Processing**: Uses `pandas` to instantly read and process thousands of rows of business data from Excel spreadsheets.
- **Deep Analytical Context**: Automatically calculates critical KPIs—Summary Statistics, Regional Performance, and Product Performance.
- **AI-Driven Insights**: Utilizes the lightning-fast Groq API running `Llama-3.3-70b-versatile` to digest the analytical context and generate meaningful business narratives.
- **Professional Formatting**: Outputs the final report in beautifully formatted Markdown (`report.md`), complete with an Executive Summary, Regional Breakdown, Product Performance, and Actionable Strategic Recommendations.
- **Sample Data Generation**: Comes with a built-in mock data generator to instantly test and showcase the agent's capabilities without needing your own proprietary data.

## 🛠️ Getting Started

### 1. Prerequisites
Ensure you have Python 3.8+ installed on your system. You will also need an API key from [Groq](https://console.groq.com/keys).

### 2. Installation
Clone the repository and install the required dependencies:

```bash
git clone https://github.com/TTins1702/Excel_AutoReport.git
cd Excel_AutoReport

# Optional: Create a virtual environment
python -m venv venv
.\venv\Scripts\activate   # On Windows
# source venv/bin/activate  # On macOS/Linux

# Install dependencies
pip install -r requirements.txt
```

### 3. Configuration
Create a `.env` file in the root directory and add your Groq API Key:

```env
GROQ_API_KEY=your_groq_api_key_here
```

### 4. Usage

To run the full automated reporting pipeline, simply execute:

```bash
python main.py --generate-sample
```

**Command-line arguments:**
- `--file <path>`: Specifies the path to the target `.xlsx` Excel file. (Default: `data/sample_sales.xlsx`)
- `--output <path>`: Specifies the output path for the Markdown report. (Default: `report.md`)
- `--generate-sample`: Automatically generates a fake dataset of 500 rows before analysis.
